# rice-swim-instructor-portal

Scaled-down prototype for a Rice Recreation private swim lesson tracking website.

## What it includes

- Instructor-side lesson entry form
- Admin-side dashboard with filters and summaries
- Local browser persistence via `localStorage`
- CSV export for simple reporting
- Rice Recreation-inspired visual design

## Production next steps

- Connect to the `Private Instructors` calendar
- Add Google auth and role-based access
- Store records in a shared database instead of the browser
- Add billing, package tracking, and payroll reconciliation
